# 習題 3 -- 請用 AI 為 nn0.py 寫機器學習或神經網路學習範例

這個習題我用了PyTorch實作五種常見的神經網路模型，以了解不同模型的結構與適用情境。分別為線性回歸、多層感知機、多分類神經網路、卷積神經網路以及循環神經網路。透過實際撰寫程式與觀察資料在模型中的輸入與輸出過程，我學習到各模型在預測、分類、影像處理及序列資料分析上的應用差異，也更了解不同層級與函數在神經網路中的作用與設計原理。


---



## 5個範例核心

### 1.單層線性回歸
* **幹嘛用的**：最基礎的直線預測（$y = ax + b$），通常拿來做連續數值的預估（例如估算房價）。
* **資料流向**：丟進去的測試資料是 `[Batch=2, Features=3]`（一次丟 2 筆，每筆 3 個特徵），經過 `nn.Linear` 乘上權重並加偏置後，自動吐出 `[Batch=2, Features=1]` 的預測值。

### 2.多層感知機 (`MultiLayerPerceptron`)
* **幹嘛用的**：用來處理二分類任務（是或不是 / 0 或 1），像是垃圾郵件辨識。
* **為什麼加 ReLU**：如果只用線性層，模型只會畫直線。疊了兩層線性層並在中間塞一個 `nn.ReLU()` 非線性激活函數，能讓模型學會「轉彎」去適應複雜的資料邊界。
* **最後的 Sigmoid**：硬把最後的輸出值壓縮到 `0 ~ 1` 之間，這個數值就可以直接當成「機率」來看（例如 0.85 代表有 85% 的機率是垃圾郵件）。

### 3. 多分類神經網路 (`MultiClassClassifier`)
* **幹嘛用的**：分類多個標籤的任務（例如辨識貓、狗、鳥）。這裡改用 `nn.Sequential` 把所有層像樂高一樣串起來，讓 `forward` 只要一行 `return self.network(x)` 就結束，畫面超乾淨。
* **🚨 這裡有大盲區（超多人踩雷）**：
  最後輸出的形狀是 `[Batch, 3]`。你可能會想說最後不用加 Softmax 轉換成機率嗎？**絕對不要加**。因為 PyTorch 內建的 `nn.CrossEntropyLoss`（交叉熵損失函數）在計算時會自動幫你補上 `LogSoftmax`。如果你在網路最後一層自己手動加了，等於重複計算，訓練時梯度會直接爛掉。

### 4. 卷積神經網路 (`SimpleCNN`)
* **幹嘛用的**：處理影像分類（影像維度通常是 `[Batch, 通道, 高, 寬]`）。
* **結構邏輯**：用 `nn.Conv2d` 去抓空間特徵，再用 `nn.MaxPool2d` 把圖片大小砍半降維。最後用 `torch.flatten` 把特徵圖拉平成一維向量，直接接上前面提到的 `nn.LazyLinear` 自動對齊分類輸出，徹底告別手算維度。

### 5. 循環神經網路 (`SimpleRNN`)
* **幹嘛用的**：處理有前後時間順序的序列資料（文字、股票走勢）。這裡設定 `batch_first=True` 比較貼近大家日常的資料習慣 `[Batch, 序列長度, 特徵數]`。
* **關鍵操作 `[:, -1, :]`**：
  資料送進 RNN 後，吐出來的 `rnn_out` 形狀是 `[Batch, 5, 8]`（代表長度為 5 的序列中，每個時間步都產生了一個 8 維的隱藏狀態）。但做序列分類或預測時，我們通常只在乎「讀完整句話（最後一個時間步）」之後模型腦袋裡留下的記憶。所以用 `rnn_out[:, -1, :]` 把最後一個時間步的特徵切出來，再送進全連接層做出最終判斷。
