# 習題 3 -- 請用 AI 為 nn0.py 寫機器學習或神經網路學習範例

參考AI的範例發想，然後依我的理解打了五個範例的README.md。  
這個習題我用了PyTorch實作五種常見的神經網路模型，以了解不同模型的結構與適用情境。分別為線性回歸、多層感知機、多分類神經網路、卷積神經網路以及循環神經網路。透過實際撰寫程式與觀察資料在模型中的輸入與輸出過程，我學習到各模型在預測、分類、影像處理及序列資料分析上的應用差異，也更了解不同層級與函數在神經網路中的作用與設計原理。


---



## 5個範例

### 1.單層線性回歸：線性回歸是最基礎的預測模型，主要用於連續數值的預估，例如房價預測。在這個範例中，我將形狀為 [Batch=2, Features=3] 的測試資料輸入模型，代表一次輸入 2 筆資料且每筆資料包含 3 個特徵，接著透過 nn.Linear 對輸入資料進行權重運算與偏置調整，最後輸出形狀為 [Batch=2, Features=1] 的預測結果，得到每筆資料對應的一個數值預測。

### 2.多層感知機：常用於二分類任務，例如垃圾郵件辨識。模型在多個線性層之間加入 nn.ReLU() 非線性激活函數，使其不再只能學習單純的線性關係，而能處理較複雜的資料特徵與分類邊界。最後透過 Sigmoid 函數將輸出值壓縮至 0 到 1 之間，作為預測結果的機率值，以判斷資料屬於某一類別的可能性。

### 3.多分類神經網路：用於處理多類別分類任務，例如辨識貓、狗或鳥等不同標籤。在這個範例中我使用 nn.Sequential 將各層依序串接，使模型結構更簡潔，並在 forward 中直接回傳整個網路的輸出。需要特別注意的是，模型最後輸出的形狀為 [Batch, 3]，不需要額外加入 Softmax 函數將結果轉換為機率，因為 PyTorch 的 nn.CrossEntropyLoss 已內建 LogSoftmax 的計算。若重複加入 Softmax，反而可能影響梯度更新，降低模型訓練效果。

### 4.卷積神經網路：主要用於影像分類任務，其輸入資料的維度通常為 [Batch, 通道, 高, 寬]。在這個範例中，我先利用 nn.Conv2d 擷取影像中的空間特徵，再透過 nn.MaxPool2d 降低特徵圖的尺寸以減少計算量。接著使用 torch.flatten 將多維特徵圖展平成一維向量，最後接上 nn.LazyLinear 完成分類輸出，使模型能自動推斷輸入維度，避免繁瑣的手動計算。

### 5.循環神經網路：主要用於處理具有時間順序的序列資料，例如文字內容或股票走勢分析。本範例設定 batch_first=True，使輸入資料維度為 [Batch, 序列長度, 特徵數]，較符合一般資料的排列方式。資料經過 RNN 後，輸出的 rnn_out 會保留每個時間步的隱藏狀態，而在進行序列分類或預測時，通常會取最後一個時間步的輸出 rnn_out[:, -1, :] 作為整段序列的特徵表示，再將其輸入全連接層，以產生最終的預測結果。
